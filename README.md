# job-seeker

A collection of Claude Code skills for evaluating job opportunities against a senior engineer's profile.

## Skills

### `job-evaluator`

Researches a company and produces a structured evaluation report covering:

- Employee sentiment (Glassdoor, Kununu)
- Salary benchmarks (Levels.fyi)
- Open positions (LinkedIn, Remotely.de)
- Candidate fit across stack, work model, compensation, and culture

**Usage:** Mention a company name or job posting URL in conversation. The skill triggers automatically on phrases like "evaluate this company", "should I apply here", or "what do employees say about X".

## Installation

Install the skill into Claude Code:

```bash
claude skill install job-evaluator.skill
```
